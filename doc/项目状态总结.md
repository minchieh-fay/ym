# YM线下组局平台 - 项目状态总结

## 🎯 项目目标

创建一个**平台主导的线下社交活动组局工具**，包含：
1. **Go后端服务** - API服务和业务逻辑
2. **管理后台** - 活动管理、用户管理、举报处理
3. **UniApp小程序** - 微信小程序和抖音小程序
